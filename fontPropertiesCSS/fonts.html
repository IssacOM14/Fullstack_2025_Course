1 Pixel is 1/96 of an Inch.

1 Point is 1/72 of an Inch.

1rem is 100% the size of the root element. In this case is xx-large.

1em is 100% the size of the parent element, the class container. In this case is 32pt.

This div has the 0.5 size of the div with the class container, that is 32pt. So, we have 16pt for this text.

This text has the same font-size but instead of using the em, we are using the fixed value of 16pt. Notice that is the same size as the previous text.

This div has the 0.25 size of the div with the class container, that is 32pt. So, we have 8pt for this text because this div is nested inside another half. So we can say, that using the em nested takes the value of the previous parent element, in this case, it was the previous div.
